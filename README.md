A console-based Java application that manages student records using file handling and serialization.
- Add Student
- View All Students
- Search Student by ID
- Update Student
- Delete Student
- Sort Students by Marks
- Show Topper
- Count Total Students
- Export Data to Text File

- Core Java
- File Handling
- Serialization
- ArrayList
- Exception Handling

- Student.java → Model class
- FileHandler.java → File operations
- Main.java → Menu-driven interface
- students.dat → Stored data file

javac *.java
java Main
